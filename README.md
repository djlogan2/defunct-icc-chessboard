# chessboard

> chessboard library

## Install

```bash
npm install --save chessboard
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'chessboard'
import 'chessboard/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
